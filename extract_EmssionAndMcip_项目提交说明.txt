项目提交数据提取脚本说明
=======================

## 功能概述
该脚本用于从CMAQ Daily COMBINE ACONC文件中提取指定变量并保存为新的nc文件，专门用于项目提交。脚本不进行任何计算（如超标天数、平均值等），仅提取原始数据并保持原始文件的属性和结构。

## 提取的变量
- O3_MDA8: O3日最大8小时平均浓度
- PM25_TOT: PM2.5总浓度
- SFC_TMP: 地面温度
- SOL_RAD: 太阳辐射
- PBLH: 边界层高度

## 脚本特点
1. 保持原始nc文件的所有全局属性
2. 保留原始文件的维度结构
3. 输出为标准nc格式，便于项目提交使用
4. 不进行任何数据计算或统计处理
5. 只提取指定的变量，减少文件大小

## 使用方法
直接运行脚本：
```bash
python extract_EmssionAndMcip_项目提交.py
```

## 配置
用户可以根据需要修改脚本中的FILE_CONFIGS列表，添加或修改输入输出文件路径。
