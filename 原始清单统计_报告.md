# 原始清单生成与统计报告

## 概览
- 目的：对广东区域原始清单（Shapefile）进行总量统计，并按 1 月与 7 月比例分配到千吨月值，同时输出对比图。
- 范围：2000 年与 2023 年两套原始清单数据。

## 使用脚本
- 总量统计：读取各部门 Shapefile 并汇总污染物总量（单位：吨），输出年度总量（千吨）
  - 脚本： [GuangDong/emissionlist/read_shapefile_totals.py](GuangDong/emissionlist/read_shapefile_totals.py)
  - 说明：遍历 `GD2000_原始清单` 与 `GD2023_原始清单` 下各部门 `.shp`，统计字段 `SO2/NOX/CO/VOC/NH3/PM10/PM25/BC/OC` 的总和。
  - 输出：每年的 `totals.json`（含 `overall_kilotons` 年度总量，单位：千吨）
    - [GuangDong/emissionlist/GD2000_原始清单/totals.json](GuangDong/emissionlist/GD2000_原始清单/totals.json)
    - [GuangDong/emissionlist/GD2023_原始清单/totals.json](GuangDong/emissionlist/GD2023_原始清单/totals.json)

- 比例分配：根据逐月 CSV 计算 1 月/7 月占全年比例，并将年度千吨总量按比例分配到两个月
  - 脚本： [GuangDong/emissionlist/原始清单统计_比例计算.py](GuangDong/emissionlist/原始清单统计_比例计算.py)
  - 说明：
    - 读取 `EM_YYYYMM_ALL_OnlyGuangDong.csv` 计算每污染物的月总量与全年总量（吨）。
    - 读取 `totals.json` 的 `overall_kilotons` 年度总量（千吨）。
    - 使用月占比将年度千吨总量分配到 1 月与 7 月，并附上全年行。
  - 输出：
    - 比例表（1月/7月占全年比例）： [GuangDong/emissionlist/GuangDong_MonthShare_2000_2023.csv](GuangDong/emissionlist/GuangDong_MonthShare_2000_2023.csv)
    - 分配后千吨表（含 1月、7月、全年）： [GuangDong/emissionlist/GuangDong_2000_2023_原始清单排放量千吨.csv](GuangDong/emissionlist/GuangDong_2000_2023_原始清单排放量千吨.csv)

- 绘图：从分配后千吨 CSV 生成月度（1月/7月）与年度（全年）对比柱形图
  - 脚本： [GuangDong/BarCharts_peopleoutput_CN_AnnualAndMonth_原始清单.py](GuangDong/BarCharts_peopleoutput_CN_AnnualAndMonth_原始清单.py)
  - 说明：读取 `GuangDong_2000_2023_原始清单排放量千吨.csv`，绘制 2000/2023 的对比图；标题为“CMAQ模拟区域{污染物}排放量”，去除网格与图例，增大字体。
  - 输出目录： [GuangDong/Emission_Comparison_BoxPlots](GuangDong/Emission_Comparison_BoxPlots)

## 输入数据
- 原始清单（Shapefile）：
  - [GuangDong/emissionlist/GD2000_原始清单](GuangDong/emissionlist/GD2000_原始清单)
  - [GuangDong/emissionlist/GD2023_原始清单](GuangDong/emissionlist/GD2023_原始清单)
- 月度汇总 CSV（广东）：文件命名为 `EM_YYYYMM_ALL_OnlyGuangDong.csv`，位于
  - [GuangDong/emissionlist](GuangDong/emissionlist)

## 输出数据
- 年度总量（千吨）：
  - [GuangDong/emissionlist/GD2000_原始清单/totals.json](GuangDong/emissionlist/GD2000_原始清单/totals.json)
  - [GuangDong/emissionlist/GD2023_原始清单/totals.json](GuangDong/emissionlist/GD2023_原始清单/totals.json)
- 比例与分配结果：
  - [GuangDong/emissionlist/GuangDong_MonthShare_2000_2023.csv](GuangDong/emissionlist/GuangDong_MonthShare_2000_2023.csv)
  - [GuangDong/emissionlist/GuangDong_2000_2023_原始清单排放量千吨.csv](GuangDong/emissionlist/GuangDong_2000_2023_原始清单排放量千吨.csv)
- 图表输出：
  - [GuangDong/Emission_Comparison_BoxPlots](GuangDong/Emission_Comparison_BoxPlots)

## 单位与约定
- Shapefile 原始数值单位：吨/年（t/y）。
- 总量统计输出：年度总量以千吨（kt）。
- 比例分配：按 1 月/7 月占全年比例，将年度千吨总量分配到两个月份（单位仍为千吨）。
- PM2.5 在 `totals.json` 中对应键为 `PM25`，脚本已做映射处理。

## 运行示例
- 统计 2000 与 2023 年度总量（千吨）：
```bash
python3 /DeepLearning/mnt/shixiansheng/GuangDong/emissionlist/read_shapefile_totals.py --base "/DeepLearning/mnt/shixiansheng/GuangDong/emissionlist/GD2000_原始清单"
python3 /DeepLearning/mnt/shixiansheng/GuangDong/emissionlist/read_shapefile_totals.py --base "/DeepLearning/mnt/shixiansheng/GuangDong/emissionlist/GD2023_原始清单"
```
- 计算 1月/7月占比并分配千吨：
```bash
python3 /DeepLearning/mnt/shixiansheng/GuangDong/emissionlist/原始清单统计_比例计算.py
```
- 生成柱形对比图：
```bash
python3 /DeepLearning/mnt/shixiansheng/GuangDong/BarCharts_peopleoutput_CN_AnnualAndMonth_原始清单.py
```
